## Deployment

1. Clone this repository

```bash
  git clone https://github.com/Scrum-Labs/ScrumFlow.git
```
2. Navigate into the folder

```bash
    cd ScrumFlow
```
3. Install all of the necessary dependencies

```bash
  npm install
```
4. Run the application

```bash
  npm run dev
```
